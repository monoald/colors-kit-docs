The Color Blind API can simulate different types of color blindness.

## toAchromatomaly

The **toAchromatomaly** method simulates achromatomaly for a color.

### Syntax

```js
toAchromatomaly(color)
```

### Parameters

**_color:_** A color of type [BaseColor](/colors-kit-docs/docs/typescript-colors#basecolor), The method will identify the correct format to operate.

### Return value

A color of type [BaseColor](/colors-kit-docs/docs/typescript-colors#basecolor) simulating achromatomaly.

### Examples

```js
toAchromatomaly('#42D37A') // Expected output: '#73b58d'

toAchromatomaly({ r: 45, g: 75, b: 200 }) // Expected output: { r: 64, g: 77, b: 134 }
```

## toColorBlind

The **toColorBlind** method simulates any color blindness type for a color.

### Syntax

```js
toColorBlind(color)
```

### Parameters

**_color:_** A color of type [BaseColor](/colors-kit-docs/docs/typescript-colors#basecolor), The method will identify the correct format to operate.

**_colorBlindness:_** A string of type [ColorBlindness](/colors-kit-docs/docs/typescript-colors#colorblindeness).

### Return value

A color of type [BaseColor](/colors-kit-docs/docs/typescript-colors#basecolor) simulating the selected type of color blindness.

### Examples

```js
toColorBlind('#f4fa53', 'tritanopia') // Expected output: '#f49bb2'

toColorBlind({ r: 195, g: 144, b: 244 }, 'deuteranopia') // Expected output: { r: 175, g: 179, b: 213 }
```
